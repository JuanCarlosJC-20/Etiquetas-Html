Etiquetas estructurales

Estas definen la estructura general de una página HTML.

  • <html>

    Esta etiqueta es la raíz de un documento HTML. Todo el contenido de la página debe estar dentro de esta etiqueta. Define el inicio y el final del documento HTML.

  • <head>

    Esta etiqueta define el encabezado del documento HTML. Contiene información sobre el documento como el título, metadatos, enlaces a hojas de estilo y scripts.

  • <body>

    Esta etiqueta define el cuerpo del documento HTML. Contiene todo el contenido visible de la página web, como texto, imágenes, enlaces y otros elementos.

  • <header>

    Esta etiqueta define el encabezado de una sección o página HTML. Puede contener elementos como el título, el logo, el menú de navegación y otros elementos que se repiten en varias páginas.

  • <nav>

    Esta etiqueta define una sección de navegación dentro de una página HTML. Se utiliza para agrupar elementos de navegación como enlaces a otras páginas o secciones de la misma página.

  • <main>

    Esta etiqueta define el contenido principal de una página HTML. Se utiliza para agrupar el contenido que es único y relevante para la página, como artículos, secciones o cualquier otro contenido principal.

  • <footer>

    Esta etiqueta define el pie de página de una sección o página HTML. Contiene información como el copyright, los datos de contacto, enlaces a políticas de privacidad y otros elementos que se repiten en varias páginas.

  • <section>

    Esta etiqueta define una sección genérica en un documento HTML. Se utiliza para agrupar contenido relacionado dentro de una página, como artículos, secciones de un artículo o cualquier otro bloque de contenido.

    Etiquetas de texto

  • <h1> a <h6>

    Estas etiquetas definen los encabezados de un documento HTML. <h1> es el encabezado más importante y <h6> es el menos importante. Se utilizan para organizar el contenido y mejorar la accesibilidad de la página.

  • <p>

    Esta etiqueta define un párrafo de texto en un documento HTML. Se utiliza para agrupar oraciones y organizar el contenido en bloques de texto.

  • <span>

    Esta etiqueta define un contenedor en línea para agrupar texto o elementos dentro de un párrafo. Se utiliza para aplicar estilos específicos a una parte del texto sin afectar a todo el párrafo.

  • <strong>

    Esta etiqueta define un texto importante o enfatizado en un documento HTML. Se utiliza para resaltar palabras o frases que tienen un significado especial.

  • <em>

    Esta etiqueta define un texto enfatizado en un documento HTML. Se utiliza para resaltar palabras o frases que tienen un significado especial, pero con un énfasis menor que <strong>.

  • <br>

    Esta etiqueta define un salto de línea en un documento HTML. Se utiliza para crear una nueva línea sin iniciar un nuevo párrafo.

  • <hr>

    Esta etiqueta define una línea horizontal en un documento HTML. Se utiliza para separar secciones o temas dentro de una página.

  • <small>

    Esta etiqueta define texto pequeño en un documento HTML. Se utiliza para mostrar texto que tiene un tamaño menor que el texto normal, como notas al pie o información secundaria.

  • <mark>

    Esta etiqueta define un texto resaltado en un documento HTML. Se utiliza para resaltar partes del texto que son relevantes o importantes para el usuario.

  • <del>

    Esta etiqueta define texto eliminado en un documento HTML. Se utiliza para mostrar texto que ha sido eliminado o corregido, generalmente con una línea tachada.

  • <sub>

    Esta etiqueta define texto en subíndice en un documento HTML. Se utiliza para mostrar texto que debe aparecer ligeramente por debajo de la línea base, como fórmulas químicas o matemáticas.

  • <sup>

    Esta etiqueta define texto en superíndice en un documento HTML. Se utiliza para mostrar texto que debe aparecer ligeramente por encima de la línea base, como exponentes o notas al pie.

Etiquetas de listas

Estas etiquetas se utilizan para crear listas de elementos en un documento HTML. Las listas pueden ser ordenadas, desordenadas o de definición, dependiendo de la etiqueta utilizada.

  • <ul>

    Esta etiqueta define una lista desordenada en un documento HTML. Se utiliza para agrupar elementos de lista que no tienen un orden específico, generalmente con viñetas.

  • <ol>

    Esta etiqueta define una lista ordenada en un documento HTML. Se utiliza para agrupar elementos de lista que tienen un orden específico, generalmente con números o letras.

  • <li>

    Esta etiqueta define un elemento de lista en un documento HTML. Se utiliza para representar cada elemento dentro de una lista ordenada o desordenada.

  • <dl>

    Esta etiqueta define una lista de definición en un documento HTML. Se utiliza para agrupar términos y sus definiciones, generalmente con un formato de término seguido de su descripción.

  • <dt>

    Esta etiqueta define un término en una lista de definición en un documento HTML. Se utiliza para representar el término que se va a definir dentro de una lista de definición.

  • <dd>

    Esta etiqueta define una descripción en una lista de definición en un documento HTML. Se utiliza para representar la descripción o definición de un término dentro de una lista de definición.

Etiquetas de formularios

Estas etiquetas se utilizan para crear formularios en un documento HTML. Los formularios permiten a los usuarios ingresar datos y enviarlos a un servidor para su procesamiento.

  • <form>

    Esta etiqueta define un formulario en un documento HTML. Se utiliza para agrupar elementos de formulario como campos de entrada, botones y otros controles que permiten a los usuarios ingresar datos.

  • <input>

    Esta etiqueta define un campo de entrada en un formulario HTML. Se utiliza para crear diferentes tipos de campos de entrada, como texto, contraseña, correo electrónico, número, fecha, entre otros.

  • <label>

    Esta etiqueta define una etiqueta para un elemento de formulario en un documento HTML. Se utiliza para asociar una etiqueta descriptiva con un campo de entrada, mejorando la accesibilidad y usabilidad del formulario.

  • <textarea>

    Esta etiqueta define un área de texto en un formulario HTML. Se utiliza para crear un campo de entrada de texto de varias líneas, permitiendo a los usuarios ingresar texto más largo o detallado.

  • <button>

    Esta etiqueta define un botón en un formulario HTML. Se utiliza para crear diferentes tipos de botones, como botones de envío, botones de reinicio o botones personalizados con funciones específicas.

  • <select>

    Esta etiqueta define un menú desplegable en un formulario HTML. Se utiliza para crear una lista de opciones de la cual los usuarios pueden seleccionar una o varias opciones.

  • <option>

    Esta etiqueta define una opción en un menú desplegable en un formulario HTML. Se utiliza para crear las diferentes opciones que los usuarios pueden seleccionar dentro de un elemento <select>.

  • <optgroup>

    Esta etiqueta define un grupo de opciones en un menú desplegable en un formulario HTML. Se utiliza para organizar las opciones dentro de un elemento <select> en grupos lógicos.

  • <fieldset>

    Esta etiqueta define un grupo de elementos relacionados en un formulario HTML. Se utiliza para agrupar campos de entrada relacionados dentro de un formulario, mejorando la organización y la accesibilidad.

  • <legend>

    Esta etiqueta define una leyenda para un grupo de elementos relacionados en un formulario HTML. Se utiliza para proporcionar una descripción o título para un conjunto de campos de entrada agrupados dentro de un elemento <fieldset>.

  • <datalist>

    Esta etiqueta define una lista de opciones predefinidas para un campo de entrada en un formulario HTML. Se utiliza para proporcionar sugerencias o autocompletar opciones a los usuarios mientras ingresan datos en un campo de entrada.

  • <output>

    Esta etiqueta define un resultado de cálculo o una salida de un formulario HTML. Se utiliza para mostrar el resultado de una operación o cálculo realizado en el formulario, como la suma de dos números o la validación de datos.

  • <progress>

    Esta etiqueta define una barra de progreso en un formulario HTML. Se utiliza para mostrar el progreso de una tarea o proceso, como la carga de un archivo o la finalización de una operación.

  • <meter>

    Esta etiqueta define una medida escalar dentro de un rango conocido en un formulario HTML. Se utiliza para mostrar una medida o valor dentro de un rango específico, como la temperatura, el nivel de batería o cualquier otra medida cuantitativa.

Etiquetas de multimedia

Estas etiquetas se utilizan para insertar y controlar contenido multimedia en un documento HTML, como imágenes, audio y video.

  • <img>

    Esta etiqueta define una imagen en un documento HTML. Se utiliza para insertar imágenes en una página web, especificando la fuente de la imagen y otros atributos como el texto alternativo y el tamaño.

  • <audio>

    Esta etiqueta define contenido de audio en un documento HTML. Se utiliza para insertar archivos de audio en una página web, proporcionando controles para reproducir, pausar y ajustar el volumen del audio.

  • <video>

    Esta etiqueta define contenido de video en un documento HTML. Se utiliza para insertar archivos de video en una página web, proporcionando controles para reproducir, pausar, ajustar el volumen y cambiar la calidad del video.

  • <source>

    Esta etiqueta define múltiples recursos multimedia para elementos de audio y video en un documento HTML. Se utiliza para especificar diferentes formatos de archivo para un mismo recurso multimedia, permitiendo que el navegador seleccione el formato compatible.

  • <track>

    Esta etiqueta define pistas de texto para elementos de audio y video en un documento HTML. Se utiliza para proporcionar subtítulos, descripciones o cualquier otro tipo de texto sincronizado con el contenido multimedia.

  • <figure>

    Esta etiqueta define contenido ilustrativo en un documento HTML. Se utiliza para agrupar elementos multimedia como imágenes, gráficos o diagramas junto con su leyenda, proporcionando una estructura semántica para el contenido visual.

  • <figcaption>

    Esta etiqueta define una leyenda para un elemento <figure> en un documento HTML. Se utiliza para proporcionar una descripción o título para el contenido ilustrativo agrupado dentro de un elemento <figure>.

  • <iframe>

    Esta etiqueta define un marco en línea en un documento HTML. Se utiliza para incrustar otro documento HTML dentro de la página actual, como un video de YouTube, un mapa de Google o cualquier otro contenido externo.

  • <embed>

    Esta etiqueta define un contenedor para contenido externo en un documento HTML. Se utiliza para incrustar contenido multimedia o interactivo, como archivos Flash, PDF o cualquier otro tipo de contenido que no sea compatible con las etiquetas de audio o video.

  • <object>

    Esta etiqueta define un contenedor para contenido externo en un documento HTML. Se utiliza para incrustar contenido multimedia o interactivo, como archivos Flash, PDF o cualquier otro tipo de contenido que no sea compatible con las etiquetas de audio o video, proporcionando una alternativa a la etiqueta <embed>.

  • <canvas>

    Esta etiqueta define un área de dibujo en un documento HTML. Se utiliza para crear gráficos dinámicos, animaciones o cualquier otro tipo de contenido visual generado mediante JavaScript.

  • <svg>

    Esta etiqueta define un contenedor para gráficos vectoriales en un documento HTML. Se utiliza para crear gráficos escalables y de alta calidad utilizando el lenguaje de gráficos vectoriales SVG (Scalable Vector Graphics).
