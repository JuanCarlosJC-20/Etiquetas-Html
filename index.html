🏗️ Etiquetas Estructurales

Estas definen la estructura general de una página HTML.

  • <html>

    Raíz de un documento HTML. Todo el contenido debe estar dentro de esta etiqueta.

  • <head>

    Encabezado del documento. Contiene metadatos, título, enlaces a CSS y scripts.

  • <body>

    Cuerpo del documento. Contiene todo el contenido visible de la página.

  • <header>

    Encabezado de una sección o página. Contiene título, logo, menú de navegación.

  • <nav>

    Sección de navegación. Agrupa enlaces a otras páginas o secciones.

  • <main>

    Contenido principal de la página. Único y relevante para cada página.

  • <footer>

    Pie de página. Contiene copyright, contacto, políticas de privacidad.

  • <section>

    Sección genérica. Agrupa contenido relacionado dentro de una página.

📝 Etiquetas de Texto

Estas etiquetas dan formato y significado semántico al texto.

  • <h1> a <h6>

    Encabezados de distintos niveles. <h1> es el más importante.

    h1 Encabezado

    h3 Encabezado

    h6 Encabezado

  • <p>

    Define un párrafo de texto.

    Esto es un párrafo de ejemplo.

  • <span>

    Contenedor en línea para aplicar estilos a partes del texto.

    Texto con span coloreado dentro.

  • <strong>

    Texto importante con énfasis fuerte (negrita semántica).

    Texto importante

  • <em>

    Texto con énfasis (cursiva semántica).

    Texto enfatizado

  • <br>

    Salto de línea sin iniciar nuevo párrafo.

    Línea 1
    Línea 2

  • <hr>

    Línea horizontal separadora de secciones.

  • <small>

    Texto de menor tamaño para notas o información secundaria.

    Texto normal texto pequeño

  • <mark>

    Resalta partes del texto relevantes para el usuario.

    Texto con parte resaltada aquí.

  • <del>

    Texto eliminado o corregido (tachado).

    Precio: $100 Ahora: $70

  • <sub>

    Texto en subíndice. Usado en fórmulas químicas o matemáticas.

    H2O — agua

  • <sup>

    Texto en superíndice. Usado en exponentes o notas al pie.

    E = mc2

📋 Etiquetas de Listas

Se utilizan para crear listas ordenadas, desordenadas o de definición.

  • <ul>

    Lista desordenada con viñetas.

    • Manzana
    • Pera
    • Uva

  • <ol>

    Lista ordenada con números o letras.

    1. Primero
    2. Segundo
    3. Tercero

  • <li>

    Elemento de lista dentro de <ul> u <ol>.

  • <dl>

    Lista de definición para términos y sus descripciones.

    HTML
    Lenguaje de marcado para páginas web.
    CSS
    Lenguaje de estilos para páginas web.

  • <dt>

    Término dentro de una lista de definición.

  • <dd>

    Descripción o definición del término en <dl>.

📬 Etiquetas de Formularios

Permiten a los usuarios ingresar y enviar datos a un servidor.

  • <form>

    Agrupa todos los elementos de un formulario.

  • <input>

    Campo de entrada (texto, password, email, número, etc.).

  • <label>

    Etiqueta descriptiva asociada a un campo de entrada.

  • <textarea>

    Área de texto multilínea.

  • <button>

    Botón para enviar, reiniciar o ejecutar acciones.

  • <select> y <option>

    Menú desplegable para seleccionar opciones.

  • <optgroup>

    Agrupa opciones dentro de un <select>.

  • <fieldset> y <legend>

    Agrupa campos relacionados con un título descriptivo.

    Datos personales

  • <datalist>

    Lista de sugerencias para un campo de entrada.

  • <output>

    Muestra el resultado de un cálculo.

  • <progress>

    Barra de progreso de una tarea.

    65%

  • <meter>

    Medida escalar dentro de un rango conocido.

    70%

🎬 Etiquetas de Multimedia

Para insertar y controlar imágenes, audio, video y gráficos.

  • <img>

    Inserta una imagen especificando la fuente y texto alternativo.

  • <audio>

    Inserta audio con controles de reproducción.

  • <video>

    Inserta video con controles de reproducción.

  • <source>

    Define múltiples formatos de fuente para audio/video.

  • <track>

    Pistas de texto (subtítulos) para audio y video.

  • <figure> y <figcaption>

    Contenido ilustrativo con su leyenda descriptiva.

  • <iframe>

    Incrusta otro documento HTML (YouTube, Google Maps, etc.).

  • <embed>

    Incrusta contenido externo (PDFs, Flash, etc.).

  • <object>

    Contenedor para contenido externo, alternativa a <embed>.

  • <canvas>

    Área de dibujo para gráficos dinámicos generados con JavaScript.

  • <svg>

    Gráficos vectoriales escalables directamente en HTML.
